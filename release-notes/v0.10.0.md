- New PostToolUse hook that validates PBIR JSON files on every Write/Edit, blocking on invalid JSON
- New PostToolUse hook (experimental) that validates TMDL structural syntax via a Rust CLI binary
- Note: validation hooks only fire on `Write`/`Edit` tool calls; programmatic modifications via `Bash` (sed, python, cat, etc.) bypass the hooks
- Added SpaceParts example semantic model to the tmdl skill
- Added complete TMDL object property reference to tmdl skill
- Updated script file location guidance in `connect-pbid` skill
- New fabric-cli plugin (moved from [fabric-cli-plugin](https://github.com/data-goblin/fabric-cli-plugin)) for working with the Power BI service and Fabric
- Added some initial context for auditing semantic models (very basic for now)
